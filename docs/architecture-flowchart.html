Project Overview

PRONUVE Water Intelligence is an autonomous multi-agent AI system that monitors municipal park irrigation water consumption across Turkish cities. Using Google ADK as the orchestration framework, 21 specialized agents collaborate to detect anomalies, predict consumption trends, optimize costs, ensure regulatory compliance, and alert officials — all with minimal human intervention.

The system addresses a critical gap: Turkey's 1,400+ municipalities manage 10,000+ public parks with mostly manual water monitoring. Late detection of leaks and over-irrigation costs billions annually. PRONUVE brings AI-powered intelligence to this domain using real data from ASKİ (Ankara Water Authority) through an active research partnership funded by TÜBİTAK.

ADK Patterns Demonstrated

SequentialAgent (×3)

Data Pipeline, Governance Pipeline, Full Pipeline — ordered stage execution

ParallelAgent (×3)

5-method anomaly detection, analysis layer, optimization layer — concurrent execution

LoopAgent (×1)

Continuous monitoring cycle with configurable intervals (daily/weekly)

Human-in-the-Loop (×1)

Critical alert approval before dispatch — Approve / Modify / Reject
